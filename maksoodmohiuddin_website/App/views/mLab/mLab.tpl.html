  • ColorTheStates v1.0.0

    ColorTheStates is an interactive mobile app that offer the ability to color or draw on the state map in a canvas and learn more about each of the 50 states. This app is designed for kids to create interest in geography education. The app is developed by Maksood Mohiuddin, inspired by Aydin and Sophia Mohiuddin
